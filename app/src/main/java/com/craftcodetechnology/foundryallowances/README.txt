Краткое описание:

Помогает определить допуски размеров и масс при помощи смартфона.

||=======||=======||=======||=======||=======||=======||=======||=======||

Полное описание:

Используйте свой смартфон для определения допуска на размер и массу литой заготовки в соответствии с ГОСТ 26645–85 и ГОСТ Р 53464-2009.

Данное приложение будет полезно для использования в работе:
- технологов литейных отделов;
- ИТР, формовщиков, сборщиков форм, разметчиков литейных цехов;
- работников ОТК;
- студентов обучающихся по специальности "Литейное производство";
- конструкторов проектирующих детали изготавливаемые литьём.

Благодаря простому интерфейсу приложение работает практически на любом устройстве (минимальная версия ОС Android 4.2 Jelly Bean), очень быстро запускается и проводит вычисления. Малый размер приложения позволяет не беспокоиться о нехватке свободного места. Рекламы нет и не будет.

Пожалуйста порекомендуйте приложение своим коллегам, студентам и знакомым - всем кому оно может пригодиться :) Поставьте оценку - так вы сделаете мир лучше :)

Приложение не может исключить использование указанных нормативных документов, а лишь позволяет быстро получить некоторую справочную информацию.
